Tafe18S103 is a repository set up to further development for the Start Finance Application.

The Start Finance application is an open source project being devloped further by the company IT Works. It is currently being developed by Bradley, Bayton and Mustafa. The IT Works Director is the product owner and also involved in the development process.

The Start Finance application was partially developed before being committed to GitHub as the original master by Bradley. Subsequently, Bayton and Mustafa were invited as collaborators. All three members of the team are now simultaneously developing the code further.

New Development is required to add extra functions into the program.

This includes fuctions to maintain:

  • Personal Information
  • Contact Details
  • Appointments
  • Shopping List


    • Each group member has been allocated responsibility to develop a single area from above.

    Each function should be able to perform the following:

    1. Create new records
    2. Update existing records
    3. Delete existing records

    Group Members and there allocated area to develop:

  • Group - Shopping List
  • Bradley - Personal Information
  • Bayton - Contact Details
  • Mustafa - Appointments
